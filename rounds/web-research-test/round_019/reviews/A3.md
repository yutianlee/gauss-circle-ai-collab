### In A1's output

- **Li-Yang Case A text.**  As noted, the line `$H\ge M^{-9}T^{4}(\log T)^{\frac{171}{140}} \quad \text{ if } M<T^{-\frac{7}{16}}$` in the local TeX is likely a transcription error; the intended range is probably $M<T^{7/16}$.  The final proof-draft should cite the corrected typeset version.
- **R5-Full shifted leg positivity.**  The proof states that for $X$ sufficiently large in the active range, $m(d)$ is positive and therefore $\ell=4m-\rho>0$.  This is plausible, but the argument should be made quantitative (e.g., $X>16D^2$ guarantees $m\ge1$).  The "bounded $X$ absorbed into the implicit constant" is standard but a short explicit check would improve rigour.
- **Fejer majorant constant.**  A1 uses the constant $1/(2H+2)$.  Vaaler's inequality is $|\psi*j_N(x)-\psi(x)|\le (2N+2)^{-1}k_N(x)$.  With $N=H$ and $k_N=K_H$, the translation is exact. The proof-draft should note that the dyadic partition may contain signed weights, but the absolute-value replacement is valid for the positive bound.  No error, but a clarification in the text.
