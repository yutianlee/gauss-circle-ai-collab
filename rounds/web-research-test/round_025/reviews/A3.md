## Possible errors or hidden assumptions

- **A1 phrasing of Li-Yang Case A.**  The text notes an ambiguity $T^{-7/16}$ vs. $T^{7/16}$ in the arXiv TeX.  The conclusion that black-box import fails does not rely on resolving this ambiguity, but the range table uses $\beta_A = \delta-49/164$ which is derived from the later application, not from the ambiguous display.  This is consistent, but the source note should be explicit.
- **A2's use of the two-sided convention in $\Sigma_r$.**  The sums $\Sigma_r$ are over $1\le|h|\le H_D$, $h\equiv r\pmod4$.  For negative $h$ the residue class is defined mod 4, e.g. $-3\equiv1\pmod4$.  This is correct, but a reader could misinterpret $h\equiv r$ as positive residues only; the explicit remark in A2 clarifies it.
- **A2's weighted $|\alpha|$ M2 Cauchy kernel.**  The A3-ready formula writes $K_{2;d_1,d_2}^{(|\alpha|)} = w_D(d_1)w_D(d_2) \sum_{odd\,h} 4|\alpha_h| e(\cdots)$.  The factor $4$ is $|C_h|^2$; note that the outer factor already includes $L=\sum|\alpha_h|$, so the Cauchy--Schwarz step should be checked for constant consistency.  The diagonal estimate $D(\log H_D)^2$ is plausible, but the exact constant depends on the normalisation of the Cauchy inequality; the diagnostic is safe at exponent level.
- **Potential hidden cancellation from $\Phi$ taper.**  Both A1 and A2 mostly treat $\alpha_h$ as $\ll 1/|h|$, ignoring the $\Phi$ taper near $H_D$.  The $\Phi$-asymmetry lemma shows $\Phi(1-u)=1-\Phi(u)$, so the weights decrease at the top end; this can only improve bounds, so no false claim results.  However, numerical implementations should use exact $\Phi$ to avoid artefacts.
- **The $X^{5/4}$ heuristic and the uniform-fiber assumption.**  A2 correctly lists the assumptions it ignores, but it also says "the $X^{5/4}$ heuristic is a warning against absolute majorization."  Without a proof that the weighted absolute mass actually attains this size, it remains a hypothetical warning; it does not falsify a route by itself.
