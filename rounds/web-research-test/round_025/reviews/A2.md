## Additional theorem-hypothesis audit

The following checks should be added before any Round 25 contribution is promoted into the proof draft.

1. **Vaaler transcription check.** A1's H4 source normalization is strong, but the proof draft should still record the exact map
$$
N\mapsto H,\qquad j_N\mapsto V_H,\qquad k_N\mapsto K_H,\qquad \widehat J\mapsto \Phi.
$$
The sign of
$$
\alpha_{h,H}=-\frac{\Phi(|h|/(H+1))}{2\pi i h}
$$
is particularly important, because changing it would switch the paired real formula from an imaginary part to its negative. A1 states the map correctly; the repository should preserve it as a source-audit lemma rather than relying on memory.

2. **Dyadic weight hypotheses.** Both A1 and A3 use $w_D(d)$ as a bounded dyadic weight. For R5, boundedness and support are enough. For any later Bombieri--Iwaniec or Li--Yang comparison, the theorem may require smoothness, compact support in a fixed interval after scaling, and bounds such as
$$
w_D^{(j)}(d)\ll_j D^{-j}.
$$
The proof draft should separate the residual lemma, which only needs $|w_D|\le1$, from theorem-import lemmas that may require smooth weights. This distinction prevents a later source theorem from being applied to a rough partition.

3. **Fourth-moment coefficient hypothesis.** A2's $\mathcal Q_4$ object uses the exact coefficients
$$
\beta_h=\alpha_{h,H_D}C_h.
$$
Any numerical binning that substitutes $1/h$ for $\alpha_{h,H_D}$ should be labeled as a proxy computation. The exact endpoint taper
$$
\Phi(|h|/(H_D+1))
$$
could change the distribution of mass near $|h|=H_D$, especially in near-collision bins that preferentially use large frequencies. A3 should therefore record both exact-weight and proxy-weight outputs.

4. **CRI convention hypothesis.** A2's two-sided CRI convention and A1's paired real convention are both valid, but they are not interchangeable. The two-sided convention has residue classes for negative $h$ modulo $4$, while the paired real convention sums only over positive odd $h$ and inserts real parts. A theorem or data table should state which convention it uses before reporting
$$
R_{\rm CRI}=\frac{|\Sigma_1-\Sigma_3|^2}{|\Sigma_1|^2+|\Sigma_3|^2}.
$$
Otherwise the same numerical value could be misread as measuring two different objects.

5. **H13 transform hypothesis.** A3's H13 check is useful only if the transform includes the amplitude, support, nonstationary tails, and endpoint behavior. A leading phase of the form $\sqrt{Xhm}$ is not enough to justify a conclusion about character survival. The exact post-transform kernel should be written before applying a norm or spacing test.

6. **Li--Yang source hierarchy.** A3's TeX audit is useful, but the proof draft should prefer the rendered arXiv or published statement when there is a mismatch. The safe current conclusion is not that every local TeX line has been resolved; it is that the available theorem ranges still do not cover the endpoint Vaaler height.

## Boundary-case audit

This review flags several boundary cases that are easy to miss because the main formulas look clean.

1. **Integer and half-integer residuals.** R5 handles exact resonances by setting $W_\Delta(0)=1$. The nearest-integer tie case $X/d=m+1/2$ should also be included explicitly. It does not create a new exponent issue, but it affects reproducible code because different nearest-integer conventions can place the same term in neighboring bins.

2. **Short blocks.** A1 states that $D<X^{1/4}$ is handled directly. The proof draft should specify whether the dyadic partition begins at $D=1$ or at a smoothed cutoff, and should show that the sum of the short blocks is
$$
\sum_{d<X^{1/4}}1\ll X^{1/4}.
$$
If a smooth partition is used, bounded overlap should be stated so that no logarithmic factor is accidentally omitted. The logarithm is harmless under $X^\epsilon$, but the bookkeeping should be explicit.

3. **Zero frequencies in fourth moments.** The original $h$-sum excludes $h=0$. In denominator-paired parameterizations, equations such as $h_2=h_1-ku^2$ can produce $h_2=0$. Such tuples are not part of the original sum and should be excluded in exact binning. A3's binning code should include assertions that every $h_i$ satisfies $1\le |h_i|\le H_D$ after parameterized construction.

4. **Small frequencies.** The $X^{5/4}$ heuristic uses a typical-frequency weight $H_D^{-4}$. If near-collisions concentrate at $|h_i|=1$ or another bounded range, then the coefficient model changes. Every near-collision table should report frequency quantiles, including $\min_i|h_i|$, $\max_i|h_i|$, and the fraction of mass using any $|h_i|\le 3$.

5. **GCD-rich denominator pairs.** In the denominator-paired family, writing $d_1=gu$, $d_3=gv$ shows that large $g$ makes $u,v$ small and permits more values of $k$. The weighted summation should therefore be organized by $g$, not just by $d_1,d_3$. A rough but useful table would report the exact-resonance mass by dyadic $g$ bins.

6. **Floating-point resonance detection.** A3's numerical recommendations should not rely on testing whether a floating-point value is exactly integral. For integer $X,d$, exact resonance of $X/d$ should be detected by integer divisibility. For shifted legs, resonance of $(X/d+\rho)/4$ should be detected by checking whether $X+d\rho$ is divisible by $4d$. This avoids false misses caused by rounding.

## Proof-draft insertion checklist

The next proof-draft pass should separate identities, inequalities, and numerical experiments into different lemma classes. A useful local convention is:
$$
\text{Identity}\quad\Longrightarrow\quad\text{no asymptotic loss},
\qquad
\text{Inequality}\quad\Longrightarrow\quad\text{named hypotheses and constants},
\qquad
\text{Experiment}\quad\Longrightarrow\quad\text{parameters and precision}.
$$
Under this convention, A1's H4 transcription belongs in the identity class after the rendered source check, R5-Full belongs in the inequality class, and A3's CRI and H13 computations belong in the experiment class. Mixing these classes would make later review harder because a numerically useful observation could be mistaken for a theorem input.

For H4, the insertable statement should include both the trigonometric polynomial and the residual majorant. The object being used is not only
$$
V_H(x)=\sum_{1\le |h|\le H}\alpha_{h,H}e(hx),
$$
but also the pointwise enclosure
$$
|\psi_C(x)-V_H(x)|\le {1\over 2H+2}K_H(x),
$$
where $\psi_C$ denotes the centered periodic sawtooth. The arithmetic identity uses $\psi_F(t)=t-\lfloor t\rfloor-1/2$, so the insertion should add the endpoint conversion as a separate displayed line. That keeps the source theorem and the arithmetic convention from being silently merged.

For R5-Full, the insertable statement should record the exact weight being bounded before any dyadic summation. A suitable local form is
$$
\sum_{d\sim D} w_D(d)K_{H_D}(X/d)
\ll_\epsilon D+H_D X^\epsilon+X^{1/4+\epsilon},
$$
with the actual repository version adjusted to the two shifted legs and to the normalized residual coefficient $(2H_D+2)^{-1}$. The audit point is that the Fejer kernel has a nonnegative Fourier expansion, so exact resonances are counted arithmetically rather than estimated by cancellation. The proof should therefore expose the divisor-count step and the role of $W_\Delta(X/d-m)$ before applying a final dyadic sum.

For M9 and M9b, the insertion should keep the raw two-sided complex form and the paired real form side by side. In one convention the coefficient sign is carried by $\alpha_{h,H_D}$ and by negative frequencies; in the other it is carried by a real or imaginary part after pairing $h$ with $-h$. This line certifies the passage to a real-valued implementation when $w_D(d)$ is real. It does not certify a theorem for complex weights or for a transformed kernel after Cauchy--Schwarz. Those variants need their own statement.

For the fourth moment, the exact resonance equation should be written before any size heuristic:
$$
N=h_1d_2d_3d_4-h_2d_1d_3d_4+h_3d_1d_2d_4-h_4d_1d_2d_3.
$$
The sign pattern above is a convention; if the conjugated factors are ordered differently, the signs change. The data table should therefore print the sign pattern together with the tuple count. The first finite taxonomy should include diagonal, pair-swapped, denominator-paired, semi-diagonal, and residual exact classes, with a column for excluded zero-frequency tuples. This prevents a parameterization from adding tuples that were absent from the original $h_i\ne0$ sum.

and verify that the additive shift does not alter the derivative nondegeneracy being invoked. If a source theorem requires a smooth amplitude and the current dyadic weight is rough, the draft should either smooth the weight with a quantified boundary cost or mark the theorem as unavailable for the present object.

For numerical certificates, the repository should ask for a reproducible table format rather than prose summaries. A minimal row is
$$
(X,D,H_D,\text{leg},\text{precision},\text{exact count},\text{near count},\text{signed mass},\text{absolute mass},\text{normalization}).

The near-resonance threshold should be printed as a rational or decimal with enough digits to reproduce bin membership. ## Verification

I have verified the following formula-level claims:

1. **Coefficient Conjugacy:** For $\alpha_{h,H} = -\frac{\Phi(|h|/(H+1))}{2\pi i h}$, if $h > 0$, $\alpha_{h,H} = \frac{i}{2\pi h}\Phi(\frac{h}{H+1})$. Then $\alpha_{-h,H} = -\frac{\Phi(|-h|/(H+1))}{-2\pi i h} = \frac{\Phi(h/(H+1))}{2\pi i h} = -i \frac{\Phi(h/(H+1))}{2\pi h} = \overline{\alpha_{h,H}}$. The identity is correct.
2. **Phi Asymmetry:** $\Phi(u) = \pi u(1-u)\cot(\pi u) + u$. Then $\Phi(1-u) = \pi(1-u)u\cot(\pi(1-u)) + 1 - u$. Since $\cot(\pi - \pi u) = -\cot(\pi u)$, we have $\Phi(1-u) = -\pi u(1-u)\cot(\pi u) + 1 - u = 1 - (\pi u(1-u)\cot(\pi u) + u) = 1 - \Phi(u)$. The identity $\Phi(u) + \Phi(1-u) = 1$ is correct, and $\Phi(1-u) = \Phi(u)$ is false except at $u=1/2$.
3. **Integer Jump Convention:** $K_H(0) = \sum_{|k|\le H} (1 - \frac{|k|}{H+1}) = 1 + 2\sum_{k=1}^H (1 - \frac{k}{H+1}) = 1 + 2(H - \frac{H(H+1)}{2(H+1)}) = 1 + 2(H - H/2) = 1 + H$. Thus $\frac{K_H(0)}{2H+2} = \frac{H+1}{2(H+1)} = 1/2$. The majorant exactly covers the half-jump.
