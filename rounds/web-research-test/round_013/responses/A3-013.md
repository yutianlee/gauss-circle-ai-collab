### 2.  R5 numerical test design (Fejer product-count residual)

R5 asserts that for $X\ge2$, $X^{1/4}\le D\le X^{1/2}$, $H\asymp D X^{-1/4}$,

$$
\frac1H\sum_{d\asymp D} |w_D(d)|\,K_H(X/d) \ll_\epsilon X^{1/4+\epsilon},
$$

and, for $\rho=1,3$,

$$
\frac1H\sum_{d\asymp D} |w_D(d)|\,K_H\!\Bigl(\frac{X/d+\rho}{4}\Bigr)
   \ll_\epsilon X^{1/4+\epsilon}.
$$

#### 2.1  Explicit computation kernels

For a concrete test we ignore the smooth weight $w_D$ (replace by $1$) and take $H_D = \big\lfloor D X^{-1/4}\big\rfloor$ (the implied constant can be $1$).  Define

$$
R_1(D;X) = \frac1{H_D}\sum_{d\in I_D} K_{H_D}\!\Bigl(\frac{X}{d}\Bigr),
\qquad
I_D = \{d\in\mathbb N : D\le d < 2D\},
$$

$$
R_{2,\rho}(D;X) = \frac1{H_D}\sum_{d\in I_D} K_{H_D}\!\Bigl(\frac{X/d+\rho}{4}\Bigr),
\qquad \rho=1,3.
$$

The Fejer kernel is evaluated as

$$
K_H(t) = \frac{1}{H+1}\Bigl(\frac{\sin\pi(H+1)t}{\sin\pi t}\Bigr)^2
      = \begin{cases}
        H+1, & \sin\pi t = 0,\\
        \text{formula above}, & \text{else}.
        \end{cases}
$$

#### 2.2  Code skeleton (Python-like)

```python
import numpy as np

def K_H(t, H):
    s = np.sin(np.pi * t)
    if abs(s) < 1e-14:
        return float(H + 1)
    return (np.sin(np.pi * (H + 1) * t) / s) ** 2 / (H + 1)

def R1(X, D):
    H = int(np.floor(D * X**(-0.25)))
    s = 0.0
    for d in range(D, 2 * D):
        s += K_H(X / d, H)
    return s / H

def R2(X, D, rho):
    H = int(np.floor(D * X**(-0.25)))
    s = 0.0
    for d in range(D, 2 * D):
        s += K_H((X / d + rho) / 4, H)
    return s / H
```

#### 2.3  Test plan and expected scaling

* **Parameters:** $X\in\{10^6,10^8\}$ (squares, near-squares, non-squares, divisor-rich).
* **Dyadic blocks:** $D = \lfloor X^{1/4}\rfloor,\ \lfloor X^{3/8}\rfloor,\ \lfloor X^{1/2}\rfloor$.
* **Output:** normalised values $R_1(D;X)/X^{1/4}$ and $R_{2,\rho}(D;X)/X^{1/4}$.
* **Prediction:** all ratios should be bounded by a small constant (e.g. $\le 5$), confirming that the residual does not exceed a constant multiple of the zero-mode $D/H_D\asymp X^{1/4}$.

### 3.  M9 fixed-coefficient main-term numerics

The M9 targets are (Leg 1 and Leg 2)

$$
\begin{aligned}
\mathcal M_1(D;X) &=
-4\sum_{1\le |h|\le H_D} \alpha_{h,H_D}
\sum_{d\in I_D} \chi_4(d)\, e\!\Bigl(\frac{hX}{d}\Bigr),\\[2mm]
\mathcal M_2(D;X) &=
\;8i\sum_{1\le |h|\le H_D} \alpha_{h,H_D}\,\chi_4(h)
\sum_{d\in I_D} e\!\Bigl(\frac{hX}{4d}\Bigr),
\end{aligned}
$$

with $I_D$ as above and

$$
\alpha_{h,H}= -\frac{\Phi(|h|/(H+1))}{2\pi i\,h},\qquad
\Phi(u)=\pi u(1-u)\cot(\pi u)+u.
$$

#### 3.1  Implementation details

Because $\alpha_{-h,H} = -\overline{\alpha_{h,H}}$ and the exponential terms are conjugates (for real $d$), the sum over positive and negative frequencies can be combined:

$$
\sum_{1\le |h|\le H_D} \alpha_{h,H_D} \, e(hX/d)
 = 2\,\Re\Bigl( \sum_{h=1}^{H_D} \alpha_{h,H_D} \, e(hX/d) \Bigr).
$$

Thus a Python implementation (using complex numbers) is straightforward.

**Leg-1 code:**

```python
def Phi(u):
    if u == 0: return 1.0
    return np.pi * u * (1 - u) / np.tan(np.pi * u) + u

def alpha_h(h, H):
    u = abs(h) / (H + 1)
    return -Phi(u) / (2j * np.pi * h)

def M1(D, X):
    H = int(np.floor(D * X**(-0.25)))
    total = 0.0j
    for h in range(1, H + 1):
        ah = alpha_h(h, H)
        inner = 0.0j
        for d in range(D, 2 * D):
            if d % 2 == 0: continue
            chi = 1 if d % 4 == 1 else -1
            inner += chi * np.exp(2j * np.pi * h * X / d)
        total += ah * inner
        total += np.conj(ah) * np.conj(inner)   # negative h contribution
    return -4 * total
```

**Leg-2** needs $\chi_4(h)$ and the $1/(4d)$ phase; similar code.

#### 3.2  Stress comparisons

* **Fixed-coefficient value:** $\mathcal M_i^{\rm fix}$.
* **Arbitrary-coefficient stress:** replace $\alpha_{h,H}$ by $e^{i\theta_h}/h$ with random $\theta_h$; compute $\mathcal M_i^{\rm arb}$.
* **$L^1$ stress:** $\mathcal L_i = \frac1{H_D}\sum_{h=1}^{H_D} \frac1h \bigl|\sum_d \chi_4(d) e(hX/d)\bigr|$.

Ratios $|\mathcal M_i^{\rm fix}| / |\mathcal M_i^{\rm arb}|$ and $|\mathcal M_i^{\rm fix}| / \mathcal L_i$ indicate whether the actual Vaaler coefficients give cancellation beyond what a generic $1/h$ weight would provide.  The conjectural target is $|\mathcal M_i^{\rm fix}| \ll_\epsilon X^{1/4+\epsilon}$.

### 4.  Line-by-line Li--Yang theorem audit

From the provided TeX source (Li--Yang, arXiv:2308.14859v2):

#### 4.1  Definition of $S$ (near `\label{definition of S}`)

$$
S := \sum_{H\le h\le 2H} g\Bigl(\frac{h}{H}\Bigr)
      \sum_{M\le m\le 2M} G\Bigl(\frac{m}{M}\Bigr)
      e\!\Bigl(\frac{hT}{M}\,F\!\Bigl(\frac{m}{M}\Bigr)\Bigr).
$$

Hypotheses: $g,G$ have bounded variation; $F\in C^3([1,2])$ satisfies

$$
C_r \ge |F^{(r)}(x)| \ge C_r^{-1}\;(r=1,2,3),\qquad
|F^{(1)}F^{(3)}-3(F^{(2)})^2| \ge C_4^{-1}.
$$

#### 4.2  Case A (label `\label{case A}`)

$$
\begin{cases}
H \ge M^{-9}T^{4}(\log T)^{\frac{171}{140}} &\text{if } M < T^{-\frac{7}{16}},\\[2mm]
H \ge M^{11}T^{-6}(\log T)^{\frac{171}{140}}  &\text{if } M > T^{\frac{9}{16}},\\[2mm]
H \le M T^{-\frac{49}{164}} .
\end{cases}
$$

For the raw Vaaler endpoint block $T=X$, $M=D\asymp X^{1/2}$, $H=H_D\asymp X^{1/4}$, we have $M\asymp T^{1/2}$ which does **not** satisfy $M < T^{-7/16}$ or $M > T^{9/16}$.  The third line becomes $H\le M T^{-49/164} \asymp X^{1/2-49/164}=X^{33/164}\approx X^{0.2012}$, while $H\asymp X^{1/4}$.  Hence **Case A does not hold**.

#### 4.3  Case B (label `\label{case B}`)

$$
\begin{cases}
M \le C_5 T^{1/2},\\[2mm]
H \le \min\!\bigl\{ M^{\frac{35}{69}}T^{-\frac{2}{23}},\; B_0 M^{\frac{3}{2}}T^{-\frac{1}{2}} \bigr\}.
\end{cases}
$$

With $M\asymp X^{1/2}$, $M^{\frac{35}{69}}T^{-\frac{2}{23}} = X^{35/138-1/23}=X^{23/138}=X^{1/6}\approx X^{0.1667}$, and $B_0 M^{3/2}T^{-1/2}\asymp X^{3/4-1/2}=X^{1/4}$.  The minimum is $X^{1/6}$, which is **smaller** than $X^{1/4}$.  Therefore **Case B also fails** for the raw endpoint block (the constant $B_0$ cannot change the asymptotic exponent).

#### 4.4  Final exponent bound

The paper eventually proves

$$
\frac{S}{H} \lesssim_\epsilon T^{\theta^*+\epsilon},\qquad \theta^*=0.314483\ldots,
$$

with the restriction $H\le M T^{-\theta^*}$ and $M\le T^{1/2}$.  The Vaaler route needs $H\le M T^{-1/4}$ and $\theta=1/4$.  The high-frequency gap is $M T^{-\theta^*} < H \le M T^{-1/4}$; for $D\asymp X^{1/2}$ this means $X^{0.1855} \lesssim H \lesssim X^{1/4}$.

#### 4.5  Compatibility of M9a/M9b with Li--Yang's $S$ form

* M9a: after splitting $d\equiv 1,3\pmod4$, the inner sum can be written with a smooth weight $G$ of bounded variation; the phase $F$ is $1/x$ or a shifted version, satisfying the derivative bounds.  **Structurally compatible**, but the theorem's parameter ranges (Case A/B and $H\le MT^{-49/164}$ etc.) are violated at the endpoint.
* M9b: splitting $h=4q+r$ yields phase $e((q+\beta)X/d)$ with $\beta\in\{1/4,3/4\}$.  This can be modelled by taking $F(x)=1/(4x+\beta)$, which still satisfies the derivative conditions.  However, the shifted frequency may alter the rational-approximation step in Bombieri--Iwaniec; **this is a theorem-extension gap** (M9b-Shift).  Also, the $h$-weight $\alpha_{4q+r}\chi_4(r)$ is of bounded variation? Not necessarily, because $\chi_4(r)$ alternates; but after residue splitting it becomes smooth in $q$.

**Overall:** black-box import of Li--Yang's theorem is impossible for the raw endpoint Vaaler block.  Structural phase similarity is not sufficient.

### 5.  C2/H13 stationary-phase uniformity

The odd-lattice Poisson transform (C2) gives

$$
S_{\rm odd}(k,D) = \sum_{\substack{d\sim D\\2\nmid d}} w_D(d) e(kX/d)
                 = \frac12\sum_{m\in\mathbb Z} I(m) - \frac12\sum_{\mu\in\mathbb Z+1/2} I(\mu),
$$

with

$$
I(\xi)=\int_{\mathbb R} w_D(u)\, e\Bigl(\frac{kX}{u}-\xi u\Bigr)\,du .
$$

Only $\xi<0$ gives a stationary point; write $\xi=-m$ ($m>0$).  Then

$$
\phi(u)= \frac{kX}{u} + mu ,\qquad
\phi'(u_0)=0 \;\Longrightarrow\; u_0=\sqrt{\frac{kX}{m}},\qquad
\phi(u_0)=2\sqrt{kXm},\qquad
\phi''(u_0)=\frac{2m}{u_0}.
$$

The leading stationary-phase expansion (C2-SP0) is

$$
I(-m) \sim \frac{w_D(u_0)}{\sqrt{2m/u_0}}\,
         e\!\Bigl(2\pi i\,2\sqrt{kXm} + \frac{\pi i}{8}\Bigr).
$$

Amplitude scale: $|I(-m)|\asymp D^{3/2}(kX)^{-1/2}$ when $m\asymp kX/D^2$.

**Two distinct large parameters** (after scaling $u=Dv$):

$$
M_{\rm dual} = \frac{kX}{D^2} \quad\text{(dual length, controls the number of effective dual frequencies)},
$$
$$
\Lambda = \frac{kX}{D} \quad\text{(large parameter in the stationary phase, enters the error as } \Lambda^{-1/2}\text{)}.
$$

#### 5.1  Regimes to be treated in C2-SPU (uniform stationary phase)

| regime | condition | required estimate |
|--------|-----------|-------------------|
| interior stationary | $m\asymp kX/D^2$, $kX/D^2\gg 1$ | standard stationary phase with error $O(\Lambda^{-1/2})$ |
| short dual length | $kX/D^2 \asymp 1$ | stationary phase not valid; direct integration-by-parts or Van der Corput |
| nonstationary | $m$ far from the stationary range | $|m|^{-N}$ decay via integration by parts (smooth weight) |
| boundary | $u_0$ near the support edge of $w_D$ | uniform expansion using Fresnel-type integrals |

A complete C2-SPU lemma must provide uniform estimates in $1\le k\le H_D$ and $X^{1/4}\le D\le X^{1/2}$ with explicit dependence on $\Lambda$ and $M_{\rm dual}$.  **This lemma is not yet proved**; the leading term is accepted.

### 6.  Signed-spacing toy tests

#### 6.1  Q1-Spectral operator-norm invariance (symbolic check)

Let $\mathcal D = \{d_1,\dots,d_n\}\subset\mathbb N_{\rm odd}$ be a short list of denominators.  Define the unsigned spacing kernel

$$
K_{ij} = \sum_{|h|\le H_D} e\!\Bigl( hX\Bigl(\frac1{d_i}-\frac1{d_j}\Bigr) \Bigr).
$$

Let $U = \operatorname{diag}(\chi_4(d_1),\dots,\chi_4(d_n))$; $U$ is real and $U^2=I$.  Then $K^\chi = U K U^*$ and

$$
\|K^\chi\|_{\rm op} = \|K\|_{\rm op}.
$$

**Toy example (hand calculation):** take $n=2$, $d_1=1,\;d_2=3$, $X$ arbitrary, $H_D=1$.  Then

$$
K = \begin{pmatrix}
2 & e^{2\pi i ( -2X/3)} \\
e^{2\pi i (2X/3)} & 2
\end{pmatrix},\qquad
U = \begin{pmatrix}1 & 0 \\ 0 & -1\end{pmatrix}.
$$

$U^* K U = \begin{pmatrix}2 & -e^{-2\pi i(2X/3)} \\ -e^{2\pi i(2X/3)} & 2\end{pmatrix}$, whose singular values are identical.  Operator norm unchanged.

This confirms that **operator-norm-only estimates cannot exploit the $\chi_4$ sign** when the character enters via diagonal unitary conjugation.

#### 6.2  Cross-Residue Interference (CRI) statistic

Define (following Round 12 proposal)

$$
S_r(h) = \sum_{m} w_D(4m+r)\, e\!\Bigl(\frac{hX}{4m+r}\Bigr),\qquad r\in\{1,3\},
$$

and

$$
\mathcal S_{\rm CRI} = \sum_{h=1}^{H_D} \alpha_{h,H_D}\; S_1(h)\,\overline{S_3(h)}.
$$

The absolute comparison is $\mathcal S_{\rm abs} = \sum_{h=1}^{H_D} |\alpha_{h,H_D}|\,|S_1(h)|\,|S_3(h)|$.  If numerical experiments show $|\mathcal S_{\rm CRI}| \ll \mathcal S_{\rm abs}$, the residual-class interference could provide cancellation not seen by operator norms.

**Code sketch:**

```python
def S_r(X, D, r, h):
    inner = 0.0j
    for m in range( (D-r)//4, (2*D-r+3)//4 ):   # integers s.t. 4m+r in [D,2D)
        d = 4*m + r
        inner += np.exp(2j*np.pi * h * X / d)
    return inner

# CRI statistic for given D, X:
H = int(D * X**(-0.25))
S_CRI = 0.0j
for h in range(1, H+1):
    ah = alpha_h(h, H)
    S_CRI += ah * S_r(X,D,1,h) * np.conj(S_r(X,D,3,h))
```

#### 6.3  M9b fractional-shift matrix test

For a short integer range $q\in[1,Q]$ and a set of odd denominators $\{d_i\}$, form

$$
A^{(\beta)}_{q,i} = e\!\bigl((q+\beta)X/d_i\bigr),\qquad \beta\in\{0,\tfrac14,\tfrac34\}.
$$

Construct Gram matrix $M^{(\beta)}_{i,j} = \sum_{q=1}^Q A^{(\beta)}_{q,i} \overline{A^{(\beta)}_{q,j}}$.  Compute its spectral radius $\rho_\beta$.  If $\rho_{1/4},\rho_{3/4}\approx \rho_0$, the fractional shift does not alter spacing geometry; if they grow, the shift may break the rational-approximation step.

**Python code:**

```python
def gram_rho(X, D, Q, beta):
    odds = [d for d in range(D, 2*D) if d%2==1]
    n = len(odds)
    M = np.zeros((n,n), dtype=complex)
    for i, d1 in enumerate(odds):
        for j, d2 in enumerate(odds):
            s = 0.0j
            for q in range(1, Q+1):
                s += np.exp(2j*np.pi * (q+beta)*X*(1/d1 - 1/d2))
            M[i,j] = s
    return np.linalg.norm(M, 2)   # spectral norm
```

### 7.  Exploratory allocation: H13 near $D\asymp X^{1/2}$

## Hidden assumptions and potential gaps

1. **Vaaler coefficient sign** -- if the true sign of $\alpha_{h,H}$ is opposite, the sign of $\mathcal M_i$ flips; cancellation arguments depending on the specific sign pattern could break.  The source must be checked.
2. **Residual majorant constant** -- the proof uses $\frac{1}{2H+2}$; if the true constant is larger (e.g. $2/H$), the product-count bound may still hold with an adjusted constant, but must be verified.
3. **Second-leg product-count** -- the congruence $X\approx d(4m-\rho)$ is correctly captured, but the proof must handle the case where the nearest integer $m$ is not unique or when $X/d+\rho$ is exactly half-integer.  The current outline uses "nearest"; a rigorous treatment requires tie-breaking rules and checking that the distance to the nearest integer is still controlled by $|X-md|/D$.
4. **M9b fractional shift** -- as noted, absorbing $\beta$ into $F$ may be legitimate, but the Bombieri--Iwaniec process uses rational approximations of $T F'(m/M)$; a fixed shift changes the set of approximating fractions and could weaken the estimate.  This is a non-trivial extension.
5. **Short dual length in C2-SPU** -- when $kX/D^2\asymp 1$, stationary phase fails; a separate bound (maybe trivial) must be supplied.  Currently only the boundary regime $D\asymp X^{1/2}, k\asymp1$ is handled (B-Boundary), but the intermediate range $k\sim D^2/X$ (which is $\gg 1$ for $D\gg X^{3/8}$) is not.
6. **Smooth dyadic weights** -- R5 and M9 rely on smooth weights to control endpoint effects; the proof draft must specify the weight functions and verify that the product-count argument remains valid with absolute values and smooth truncations.
7. **$X^\epsilon$ absorption** -- many logarithmic factors are hidden in $X^\epsilon$; for a fully rigorous bound one must track constants and replace $X^\epsilon$ by explicit powers of $\log X$; this is standard but not yet done in the proof draft.

## Counterexample or obstruction search

* **Fejer spike test with square $X$** -- when $X$ is a perfect square and $d$ is a divisor in the dyadic range, $X/d$ is an integer; $K_{H_D}(X/d)=H_D+1$, contributing $O(1)$ to $R_1(D;X)/H_D$.  The product-count argument shows the total contribution over all such $d$ is $O(X^\epsilon)$, but if $X$ has many divisors in a dyadic window, the sum could be larger.  This is a useful numerical stress test.
* **Li--Yang parameter obstruction** -- already proved that raw endpoint block fails.  However, if the M9 sum is first reduced by a different dyadic decomposition (e.g. splitting $h$ and $d$ ranges differently), could the effective $M$ and $H$ change enough to satisfy Case A or B?  This is a possible research direction.
* **Signed-spacing falsification** -- if the CRI statistic shows no improvement over absolute values, the cross-residue route is likely useless.  If the fractional-shift matrix test shows the spectral radius is insensitive to $\beta$, then M9b may be as hard as the untwisted case.  These tests are implementable and would narrow the search.

## Divergent alternatives and 20 % exploration

1. **Signed Fourier truncation** -- replace Vaaler by the signed series $\psi(t) = -\sum_{h\neq0} e(ht)/(2\pi i h)$, truncated at $H_D$.  The high-frequency tail preserves $\chi_4$ but still must be bounded; near integers it may still produce large contributions.  A quick test: bound the tail for $t$ near an integer using summation by parts and compare with Fejer residual.
2. **Gaussian-integer Hecke $L$-function route** -- express $N(\sqrt X)$ via a sum over ideals in $\mathbb Z[i]$ and use a smoothed approximate functional equation.  The oscillatory terms become sums of Kloosterman or Bessel type; the character $\chi_4$ is encoded in the $L$-function.  This is a long-term speculative direction, but a first step would be to write the exact identity connecting $P(X)$ to a contour integral of $4\zeta(s)L(s,\chi_4)$.  The Mellin--Perron approach explored earlier is essentially this route; it still requires subconvexity.

This might exploit the rational-collision sign lemma (Q1-Ext) more directly.  A toy model: treat $e(hX/d)$ as a product of two exponential sums and use a large sieve inequality with signed coefficients.  No concrete estimate exists yet.
