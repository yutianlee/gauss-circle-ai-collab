## Possible errors or hidden assumptions

**A1**

- The R5-Full proof replaces $w_D$ by $|w_D|$ when bounding the Fejer kernel. This is permissible because the kernel is non-negative, but it discards any sign structure from the original weights. For the residual this is harmless; the draft should state explicitly that the residual bound does not rely on cancellation in $w_D$.
- The second-leg shift uses $\ell=4m-\rho$ and bounds $|X-d\ell|$ by $2d$ (not $d/2$). The product-counting argument remains valid because the width is still $O(D)$ and $\Delta\asymp X^{1/4}$. The proof should note the slightly larger factor and confirm that the numerical constant remains harmless.
- The bridge theorem uses a dyadic partition of unity; the proof draft should briefly explain how the weight $w_D$ is constructed and why the smooth overlap error is absorbed into $X^\epsilon$.

**A2**

- **Q1-Spectral matrix definition.** The matrix $K$ is defined using $|\alpha_{h,H_D}|^2$. This corresponds to a Cauchy--Schwarz step that already takes absolute values on the $h$-coefficients. The diagnostic therefore applies only to methods that first form a positive-semidefinite Gram matrix. If a method estimates the signed main sum without squaring in that way, Q1-Spectral may not directly apply; the lemma should state this restriction.
- **CRI and $U^*KU$ collapse.** The cross-residue term $\operatorname{Re}\sum_h \alpha_h S_1(h)\overline{S_3(h)}$ can be rewritten as a bilinear form $\langle \mathbf{1}, B \mathbf{1}\rangle$ where $B$ is an off-diagonal block of the combined Gram matrix. The standard bound $|\langle \mathbf{1}, B \mathbf{1}\rangle|\le \|B\|_\text{op}\|\mathbf{1}\|^2$ still holds, and the operator norm of an off-diagonal block is not guaranteed to be invariant under independent sign flips; however, the current analysis does not prove that the character information is preserved after the operator-norm majorisation. A concrete numerical comparison between the signed and unsigned (absolute majorant) forms would clarify, but has not been performed.
- **C3 parity analysis for general differences.** The derivation that $\sigma(\xi_1)\sigma(\xi_2)=\frac14(-1)^{2(\xi_1-\xi_2)}$ for a shift $q=\xi_1-\xi_2$ is correct (see detailed recomputation), but the text misleadingly writes the product as $\frac14(-1)^{2q}$ without showing the $4\xi_1$ cancellation step. The missing intermediate equality could cause confusion.
- **Even-dilation parity preservation.** The statement that an even dilation preserves the parity oscillation is too vague. It requires that the dilated dual variable remain in the lattice $\frac12\mathbb Z$; otherwise the parity function $\sigma$ is not defined. The analysis should be restricted to cases where this holds, or the lemma must be revised to a weaker diagnostic.
