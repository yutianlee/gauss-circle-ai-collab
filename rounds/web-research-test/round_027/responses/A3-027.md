## Detailed reasoning

### 1.  R5 residual tables (first and shifted legs)

The Fejer kernel is

$$
K_H(t)=\sum_{|k|\le H}\Bigl(1-\frac{|k|}{H+1}\Bigr)e(kt)
        =\frac1{H+1}\Bigl(\frac{\sin\pi(H+1)t}{\sin\pi t}\Bigr)^2,
$$

with pointwise bound

$$
K_H(t)\ll\min\!\Bigl(H,\frac1{H\|t\|^2}\Bigr),\qquad
\frac1{H}K_H(t)\ll\min\!\Bigl(1,\frac1{H^2\|t\|^2}\Bigr).
$$

For a dyadic denominator block $d\asymp D$ and local Vaaler height $H_D\asymp D X^{-1/4}$, define

$$
\Delta = \frac{D}{H_D}\asymp X^{1/4},\qquad
W_\Delta(u)=
\begin{cases}
1, & u=0,\\[2pt]
\min\!\bigl(1,\Delta^2/u^2\bigr), & u\neq0.
\end{cases}
$$

**Leg 1** ($K_H(X/d)$): choose $m$ nearest to $X/d$. Then

$$
\Bigl\|\frac Xd\Bigr\| = \frac{|X-md|}{d}\asymp\frac{|X-md|}{D},
\qquad
\frac1{H_D}K_{H_D}\!\Bigl(\frac Xd\Bigr)\ll W_\Delta(X-md).
$$

Grouping by $n=md$ yields a divisor bound:

$$
\frac1{H_D}\sum_{d\asymp D}|w_D(d)|K_{H_D}\!\Bigl(\frac Xd\Bigr)
\ll \sum_{n\asymp X}\tau(n)W_\Delta(X-n).
$$

**Leg 2** ($\rho=1,3$): near-integrality of $\frac{X/d+\rho}{4}$ is equivalent to $X\approx d(4m-\rho)$. Set $\ell=4m-\rho$; then $\ell\equiv-\rho\pmod4$ and the same product-counting argument applies.

**Pseudocode for a single block**

```python
def r5_block(X, D, H, w, rho=None):
    """
    w : callable, |w(d)| <= 1, supported on d ~ D.
    Returns normalized residual = (1/H) * sum_d |w(d)| K_H(t) / X^{1/4}.
    """
    delta = D / H                 # ~ X^{1/4}
    total = 0.0
    for d in range(int(D), int(2*D)+1):
        if rho is None:
            t = X / d
        else:
            t = (X/d + rho) / 4.0
        # exact-resonance cap
        if is_integer(t):
            k_val = H + 1
        else:
            # high-precision Fejer
            s = math.pi * (H+1) * t
            k_val = (math.sin(s) / math.sin(math.pi*t))**2 / (H+1)
        total += abs(w(d)) * k_val / H
    return total / (D * X**0.25)   # normalized per dyadic block
```

The script must use **high precision** (e.g. `mpmath` with 50--100 digits) and detect integer arguments exactly.

### 2.  Raw-vs-paired M9 regression

The **raw two-sided** definitions are

$$
\mathcal M_1(D;X) = -4\sum_{1\le |h|\le H_D}\alpha_{h,H_D}
\sum_{d\asymp D}\chi_4(d)w_D(d)e(hX/d),
$$

$$
\mathcal M_2(D;X) = 4\sum_{1\le |h|\le H_D}\alpha_{h,H_D}
\bigl(e(h/4)-e(3h/4)\bigr)\sum_{d\asymp D}w_D(d)e(hX/(4d)).
$$

The **paired real** forms (valid only for real $w_D$):

$$
\mathcal M_1(D;X) = \frac4\pi\sum_{h=1}^{H_D}
\frac{\Phi(h/(H_D+1))}{h}
\operatorname{Im}\Bigl(\sum_{d}\chi_4(d)w_D(d)e(hX/d)\Bigr),
$$

$$
\mathcal M_2(D;X) = -\frac8\pi\sum_{\substack{1\le h\le H_D\\2\nmid h}}
\frac{\Phi(h/(H_D+1))}{h}
\chi_4(h)\operatorname{Re}\Bigl(\sum_{d}w_D(d)e(hX/(4d))\Bigr).
$$

**Pseudocode**

```python
def alpha(h, H):
    u = abs(h) / (H+1)
    Phi = math.pi*u*(1-u)*1/math.tan(math.pi*u) + u
    return -Phi / (2j * math.pi * h)

def M1_raw(X, D, H, w):
    total = 0j
    for h in range(-H, H+1):
        if h == 0: continue
        A = sum(chi4(d)*w(d)*cmath.exp(2j*math.pi*h*X/d) for d in D_range)
        total += alpha(h, H) * A
    return -4 * total

def M1_paired(X, D, H, w):
    total = 0.0
    for h in range(1, H+1):
        A = sum(chi4(d)*w(d)*cmath.exp(2j*math.pi*h*X/d) for d in D_range)
        total += (Phi(h/(H+1))/h) * A.imag
    return (4/math.pi) * total
```

Regression requires at least $\delta=1/4,3/8,1/2$ and several $X$ types (square, near-square, nonsquare, divisor-rich). Real-weight tests must show agreement to within round-off; complex-weight tests must **fail** the paired formulas.

### 5.  Weighted and unweighted $\mathcal M_2$ Cauchy kernels

Let

$$
C_h = e(h/4)-e(3h/4),\qquad
|C_h|^2 =
\begin{cases}
4, & 2\nmid h,\\
0, & 2\mid h.
\end{cases}
$$

**Weighted kernel** ($|\alpha_h|$-weighted):

$$
K_{2;d_1,d_2}^{(|\alpha|)} = w_D(d_1)w_D(d_2)
\sum_{1\le |h|\le H_D} |\alpha_{h,H_D}|\,|C_h|^2\,
e\Bigl(\frac{hX}{4}\bigl(\frac1{d_1}-\frac1{d_2}\bigr)\Bigr).
$$

**Unweighted kernel**: omit the $|\alpha_h|$ factor, giving diagonal scale $\asymp D H_D$.

For each kernel report:

* diagonal mass $\sum_d |K_{d,d}|$;
* off-diagonal signed mass $\sum_{d_1\neq d_2} K_{d_1,d_2}$;
* operator norm $\|K\|_{\text{op}}$;
* absolute majorant (sum over absolute values);
* signed/unsigned comparison (off-diagonal).

For the weighted kernel, **verify sign loss**: the amplitude factor contains only $|C_h|^2$, hence no $\chi_4(h)$ sign survives; the kernel is odd-$h$ restricted and character-blind. This is a bounded-scope diagnostic, not a global no-go theorem.

### 10.  Li--Yang Case A ambiguity (source audit)

The TeX source defines Case A as

```latex
\begin{cases}
H\ge M^{-9}T^{4}(\log T)^{\frac{171}{140}}  \quad  &\text{ if }M<T^{-\frac{7}{16}},\\
 H\ge M^{11}T^{-6}(\log T)^{\frac{171}{140}}  \quad &\text{ if }M>T^{\frac{9}{16}},\\

\end{cases}
```

The branch $M<T^{-7/16}$ would be vacuous for large $T$ because $M\ge 1$ while $T^{-7/16}\ll 1$. Hence the intended first branch is

$$
H\ge M^{-9}T^{4}(\log T)^{\frac{171}{140}}\quad\text{if}\quad M<T^{\frac{7}{16}}.
$$

The typo is a **misplaced minus sign** in the TeX. This correction does **not** affect the earlier guardrail: the audited height restrictions still place the endpoint Vaaler height $H_D\asymp D X^{-1/4}$ outside the reach of the final Li--Yang exponent $\theta^*$.

### 11.  H13 endpoint test (one test near $D\asymp X^{1/2}$)

Apply B-process-first (Poisson modulo 4) to the spatial-character sum of $\mathcal M_1$. Under the convention $\phi_n(u)=hX/u - nu/4$, the active stationary branch is $n=-m<0$ with

$$
u_0 = 2\sqrt{\frac{hX}{m}},\qquad
\Phi_{\text{dual}} = \sqrt{hXm},\qquad
m\asymp \frac{hX}{D^2}.
$$

At $D\asymp X^{1/2}$ and maximal $h\asymp H_D\asymp X^{1/4}$ we have $m\asymp X^{1/4}$, so the transform is nearly balanced. Build the finite matrix $K^{\text{H13}}$ using the stationary-phase amplitudes (including the Gauss factor $2i\chi_4(m)$ and the Vaaler weights). Compute the signed quadratic form, its unsigned counterpart, and the operator-norm comparator derived from the diagonal-unitary conjugation. If the signed statistic does not differ significantly from the blind norm, H13 is sign-blind at the very first post-transform step.

**Pseudocode**

```python
def H13_endpoint_kernel(D, X, H, w, m_range):
    K = np.zeros((len(m_range), len(m_range)), dtype=complex)
    for i, m1 in enumerate(m_range):
        for j, m2 in enumerate(m_range):
            total = 0j
            for h in h_range:
                # stationary amplitude for each (h, m)
                amp = stationary_amplitude(D, h, m1, m2, w)
                # Gauss factor and Vaaler weight
                coeff = alpha(h, H) * chi4(h) * chi4(m1) * chi4(m2)
                phase = math.sqrt(X * h * m1) - math.sqrt(X * h * m2)
                total += coeff * amp * cmath.exp(2j*math.pi*phase)
            K[i,j] = total
    return K
```

Compare signed statistic, unsigned statistic, and operator norm.

## Hidden assumptions

* The pointwise Fejer bound used in R5 involves an implicit absolute constant; the constant is harmless for the exponent but must be checked against Vaaler's paper to avoid a factor that could accumulate over dyadic blocks.
* The paired real formulas are proved only for real dyadic weights. Complex weights require raw two-sided definitions.
* The Li--Yang theorem requires the weights $g,G$ to be of bounded variation. The Vaaler coefficient weights $\alpha_{h,H_D}$ are not of bounded variation on the whole dyadic interval, but after splitting into dyadic blocks they become smooth and bounded-variation-type; this transition must be verified before claiming a strict import.
* The M9b shifted-$F$ formulation uses functions $F_{\rho,D}$ that depend on $D$ and $X$; a candidate theorem must allow such parameter-dependent additive shifts uniformly.
* The fourth-moment heuristic ignores arithmetic clustering (gcd structure) and sign cancellations; its predictions are hypotheses to be tested, not theorems.
* The H13 stationary-phase model assumes that all contributions outside the stationary range are negligible; a uniform lemma covering support-boundary and short-dual-length regimes is still missing.
* The cross-residue statistic CRI is a falsification metric; it is not sufficient unless paired with a bound on $|\Sigma_1|^2+|\Sigma_3|^2$ and favourable sign information.

## Counterexample

* **Exact Fejer resonance**: cases where $X/d$ or $(X/d+\rho)/4$ are integers. Verify that implementations use $K_H(0)=H+1$ and that the cap $W_\Delta(0)=1$ bounds $H^{-1}K_H(0)$ by $1+1/H$.
* **Divisor-rich $X$**: choose $X$ with many divisors (e.g. factorial or highly composite) to stress the product-count multiplicity. If the normalized residual exceeds $X^{1/4+\epsilon}$ by a large margin, the R5 proof would need tighter constant control.
* **Adversarial coefficient stress tests**: for M9, compare actual Vaaler coefficients against systematic sign patterns (e.g. signs that mimic the phase of the exponential sum) to detect whether the taper provides cancellation beyond magnitude.
* **Denominator-paired singularities**: isolate configurations $h_1=ac,\, h_3=-bc$ with small $|\Delta|$; if these carry a disproportionate fraction of the weighted mass, the fourth-moment target must be refined.

   Replace the Vaaler majorant by the finite signed Fourier series
$$

Both directions are **not** proposed as replacements for the main route; they serve as comparison modules that may illuminate the central M9 bottleneck.
