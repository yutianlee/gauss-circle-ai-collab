1. **[PROVED] Vaaler Coefficient Conjugacy:** The relation $\alpha_{-h,H} = \overline{\alpha_{h,H}}$ holds for the exact Vaaler coefficients.
   *Reasoning:* The coefficient is defined as $\alpha_{h,H} = -\frac{\Phi(|h|/(H+1))}{2\pi i h}$. Since $\Phi(u)$ is real-valued for $u \in (0,1)$, substituting $-h$ yields $\alpha_{-h,H} = -\frac{\Phi(|-h|/(H+1))}{2\pi i (-h)} = \frac{\Phi(|h|/(H+1))}{2\pi i h}$, which is exactly the complex conjugate of $\alpha_{h,H}$.

2. **[PROVED] Asymmetry of the $\Phi$ Function:** The identity $\Phi(1-u) = \Phi(u)$ is false for $u \in (0,1) \setminus \{1/2\}$.
   *Reasoning:* The function is $\Phi(u) = \pi u(1-u)\cot(\pi u) + u$. Evaluating at $1-u$ gives $\Phi(1-u) = \pi(1-u)u\cot(\pi(1-u)) + 1 - u$. Since $\cot(\pi(1-u)) = -\cot(\pi u)$, we have $\Phi(1-u) = -\pi u(1-u)\cot(\pi u) + 1 - u$. Adding the two yields $\Phi(u) + \Phi(1-u) = 1$. Thus, $\Phi(1-u) = \Phi(u)$ implies $\Phi(u) = 1/2$, which only occurs at $u = 1/2$.

3. **[PROVED] Floor-Compatible Endpoint Conversion:** The Fejer residual majorant exactly covers the discrepancy between Vaaler's centered sawtooth and the floor-compatible sawtooth at integer arguments.
   *Reasoning:* At $t \in \mathbb{Z}$, the centered Vaaler polynomial evaluates to $V_H(t) = 0$. The floor-compatible sawtooth is defined as $\psi_F(n) = -1/2$. The Fejer kernel evaluates to $K_H(0) = H+1$. The residual majorant is $\frac{1}{2H+2}K_H(0) = \frac{H+1}{2H+2} = \frac{1}{2}$. This exactly matches $|\psi_F(n) - V_H(n)| = |-1/2 - 0| = 1/2$.

## Possible errors or hidden assumptions

1. **Floating-Point Instability Near Fejer Resonances:**
   *Assumption:* Numerical implementations can evaluate $K_H(t) = \frac{1}{H+1} \left( \frac{\sin \pi(H+1)t}{\sin \pi t} \right)^2$ using standard floating-point arithmetic.
   *Error Risk:* As $t \to n \in \mathbb{Z}$, both the numerator and denominator approach zero. Standard double-precision evaluation of `sin(x)/sin(y)` can suffer unstable cancellation or return `NaN`.
   *Correction:* Implementations should use exact rational/modular detection. If $\|t\| < \epsilon_{\rm mach}$, the code should branch to the exact limit $K_H(n) = H+1$.

2. **Real Weights Assumption in Paired M9 Formulas:**
   *Assumption:* The paired real formulas for $\mathcal M_1$ and $\mathcal M_2$ are universally equivalent to the raw complex definitions.
   *Error Risk:* The derivation of the paired formulas relies on $A_{-h}(D;X) = \overline{A_h(D;X)}$, which requires real-valued dyadic weights $w_D(d)$. If a complex-valued partition of unity or an asymmetric frequency cutoff is introduced, the paired formulas will yield incorrect results.
   *Correction:* The paired formulas should be explicitly labeled as implementation lemmas valid only for real weights.

3. **Li--Yang Case A Typographical Ambiguity:**
   *Assumption:* The TeX source condition $M < T^{-7/16}$ is the intended mathematical hypothesis.
   *Error Risk:* As noted by A3, the subsequent argument in the Li--Yang paper may use $T^{7/16}$. If the condition is actually $M < T^{7/16}$, the boundary of Case A shifts significantly. At $T=X$ and $M=X^{1/2}$, $X^{1/2}$ is not less than $X^{7/16}$, meaning the endpoint block might fall entirely outside Case A regardless of the height $H$.
   *Correction:* The typeset PDF should be consulted to resolve the sign of the exponent.

4. **Positivity of the Shifted Complementary Factor:**
   *Assumption:* In the R5-Full proof for the shifted legs, the complementary factor $\ell = 4m - \rho$ is positive in the active range.
   *Error Risk:* For very small $X$ or near the upper boundary $d \approx 2X^{1/2}$, the nearest integer $m$ to $(X/d + \rho)/4$ could theoretically be zero if $X/d$ is small enough. If $m=0$, then $\ell = -\rho < 0$, breaking the divisor-counting logic which assumes positive integers.
   *Correction:* The proof draft needs to explicitly establish the lower bound for $X/d$ in the active range $D \le X^{1/2}$ to guarantee $m \ge 1$, and absorb any finite number of small-$X$ exceptions into the implied constant.

## Counterexample or obstruction search

### The M2 Cauchy--Schwarz Normalization Obstructions

The second main term is:
$$
\mathcal M_2(D;X) = 4 \sum_{1 \le |h| \le H_D} \alpha_{h,H_D} C_h \sum_{d \asymp D} w_D(d) e\left(\frac{hX}{4d}\right),
$$
where $C_h = e(h/4) - e(3h/4)$. We analyze the obstructions to standard Cauchy--Schwarz routes.

**Obstruction 1: Unweighted $h$-Cauchy Diagonal Blowup**
If we apply Cauchy--Schwarz over $h$ without extracting the $|\alpha_h|$ weights into the scalar factor, we bound $|\mathcal M_2|^2$ by:
$$
16 \left( \sum_{1 \le |h| \le H_D} 1 \right) \left( \sum_{1 \le |h| \le H_D} |\alpha_{h,H_D}|^2 |C_h|^2 \left| \sum_d w_D(d) e\left(\frac{hX}{4d}\right) \right|^2 \right).
$$
The scalar factor is $2H_D \asymp D X^{-1/4}$. The diagonal of the squared $d$-sum occurs when $d_1 = d_2$. The diagonal contribution to the kernel is $\sum_h |\alpha_h|^2 |C_h|^2 \asymp 1$. Summing over $d \asymp D$ gives a diagonal sum of $D$. Multiplying by the scalar factor, the total diagonal contribution is:
$$
\mathcal S_{\rm diag}^{({\rm unweighted})} \asymp H_D \cdot D \asymp D^2 X^{-1/4}.
$$
At the endpoint $D \asymp X^{1/2}$, this diagonal size is $(X^{1/2})^2 X^{-1/4} = X^{3/4}$. The required squared target for $\mathcal M_2 \ll_\epsilon X^{1/4+\epsilon}$ is $X^{1/2+\epsilon}$. Since $X^{3/4} \gg X^{1/2+\epsilon}$, the unweighted normalization fails on the diagonal.

**Obstruction 2: Weighted $h$-Cauchy Character Blindness**
To fix the diagonal, we extract $|\alpha_h|$ into the scalar factor:
$$
|\mathcal M_2|^2 \le 16 \left( \sum_{1 \le |h| \le H_D} |\alpha_{h,H_D}| \right) \left( \sum_{1 \le |h| \le H_D} |\alpha_{h,H_D}| |C_h|^2 \left| \sum_d w_D(d) e\left(\frac{hX}{4d}\right) \right|^2 \right).
$$
The scalar factor is $L = \sum |\alpha_h| \asymp \log H_D$. The diagonal of the kernel is $\sum |\alpha_h| |C_h|^2 \asymp \log H_D$. The total diagonal contribution is $D \log^2 H_D \ll_\epsilon X^{1/2+\epsilon}$, which is acceptable.
However, the kernel now contains the factor $|C_h|^2$. Since $C_h = 2i\chi_4(h)$ for odd $h$, we have $|C_h|^2 = 4$ for odd $h$ and $0$ for even $h$. The sign of $\chi_4(h)$ is removed inside this route. The resulting off-diagonal sum is an odd-$h$ restricted reciprocal sum with nonnegative coefficients. This triggers the `M2-CS-Character-Blindness` diagnostic: the method does not exploit the character sign to gain cancellation in the off-diagonal.

**Obstruction 3: $d$-Cauchy Diagonal Blowup**
If we apply Cauchy--Schwarz over the spatial variable $d$ to preserve the $h$-character longer, the scalar factor is $\sum_d |w_D(d)|^2 \asymp D$. The squared $h$-sum produces a diagonal when $h_1 = h_2$. The total diagonal contribution is $\asymp D \cdot D = D^2$. At the endpoint $D \asymp X^{1/2}$, this is $X$, which vastly exceeds the squared target $X^{1/2+\epsilon}$.

**Conclusion:** Standard Cauchy--Schwarz routes for $\mathcal M_2$ are obstructed either by diagonal blowup or by character erasure. This necessitates the exploration of fourth-moment or open-path statistics (Alternative A).
