## Most valuable input from others

### A2

A2's most valuable contribution is the bounded-scope Q1-Spectral diagnostic. On the odd-denominator Hilbert space

$$
\mathcal V_D=\ell^2(\{d:D\le d<2D,\ 2\nmid d\}),
$$

with

$$
U=\operatorname{diag}(\chi_4(d)),
$$

one has $U^*U=I$ and therefore

$$
\|U^*KU\|_{\operatorname{op}}=\|K\|_{\operatorname{op}}.
$$

This is a useful exact linear-algebra filter: any proposed M9 proof that inserts the spatial character only as diagonal unitary conjugation and then estimates by operator norm, spectral radius, Schur/Gershgorin, Frobenius norm, or an absolute-value matrix cannot exploit $\chi_4(d)$. A2 also correctly pairs this with the restricted trace statement

$$
\operatorname{Tr}((U^*KU)^p)=\operatorname{Tr}(K^p)
$$

for pure cyclic conjugacy-invariant traces. The value is diagnostic, not route-closing.

A2's H13 discussion is useful as a falsification framework. The modulo-$4$ Poisson/B-process transform should preserve a dual $\chi_4$ factor at the formal Gauss-sum level, with dual length

$$
m\asymp \frac{hX}{D^2}.
$$

At maximal Vaaler height $h\asymp H_D\asymp D X^{-1/4}$ and $D=X^\delta$, this gives

$$
m\asymp X^{3/4-\delta},\qquad 1/4\le\delta\le1/2.
$$

Thus H13 is only roughly balanced near $D\asymp X^{1/2}$. This is a precise reason to restrict H13 experimentation to the endpoint block before spending more effort elsewhere.

A2's BSOS proposal is useful only as a candidate object for falsification. It should not be promoted as a lemma until it has: a finite matrix definition, a target inequality, a derivation showing that the target would imply a useful M9 estimate, and a comparison with the absolute majorant/operator-norm bound.

## Claims that look correct

1. **No new exponent has been proved.** Both A2 and A3 state this, and it remains the conservative conclusion.

2. **The active bottleneck is fixed-coefficient M9, not the Fejer residual.** The residual side remains provisionally controlled by H4 plus R5-Full, conditional on final H4 source normalization and dyadic bookkeeping. The main work is now

$$
\mathcal M_1(D;X),\mathcal M_2(D;X)\ll_\epsilon X^{1/4+\epsilon}
$$

with the actual Vaaler coefficients.

3. **Q1-Spectral is correct under its stated hypothesis.** If $\chi_4(d)$ enters only through $K\mapsto U^*KU$, then every unitarily invariant norm is blind to it. This is a proved diagnostic, not a general obstruction.

4. **H12 trace invariance is correct in the pure conjugacy setting.** Closed cyclic traces cannot see a diagonal unitary conjugation. The statement does not apply to open-path statistics or non-conjugacy signed forms.

5. **M9b is not automatically covered by M9a.** The safer comparison form is

$$
e(hX/(4d))(e(h/4)-e(3h/4))
=
e\left(h\left(\frac{X}{4d}+\frac14\right)\right)
-
e\left(h\left(\frac{X}{4d}+\frac34\right)\right),
$$

with

$$
F_{\rho,D}(z)=\frac1{4z}+\frac{\rho D}{4X}.
$$

The determinant check

$$
F'F'''-3(F'')^2=-\frac3{8z^6}
$$

is correct, but theorem-level applicability still requires a uniform shifted-$F$ audit.

6. **The H13 dual phase is Hessian-degenerate.** After stationary phase the effective phase is proportional to $\sqrt{Xhm}$, and

$$
\det\nabla^2\sqrt{Xhm}=0.
$$

Therefore H13 is not a route to generic full-rank two-dimensional stationary phase or generic full-rank decoupling.

### A3

1. **R5 displayed bound has a serious typo.** A3 writes a grouped bound with

$$
\max\left(1,\min\left(1,\frac{\Delta^2}{|X-n|^2}\right)\right).
$$

This is identically $1$ and would sum to roughly $X^{1+\epsilon}$, not $X^{1/4+\epsilon}$. The correct treatment is: handle exact zero-distance terms separately using $\frac1H K_H(0)\le 2$, and otherwise use

$$
\min\left(1,\frac{\Delta^2}{|X-n|^2}\right).
$$

This is likely a typographical slip because A3's prose later describes the correct cap, but it must be fixed before entering the proof draft.

2. **H13 phase constant is wrong as written.** For the displayed phase

$$
\phi_n(u)=\frac{hX}{u}-\frac{nu}{4},
$$

the active sign is $n=-m<0$ and

$$
u_0=2\sqrt{\frac{hX}{m}}.
$$

Then

$$
\phi_{-m}(u_0)
=
\frac{hX}{u_0}+\frac{mu_0}{4}
=
\sqrt{hXm},
$$

not $2\sqrt{hXm}$. The factor $2$ belongs to the odd-lattice transform with phase $kX/u+mu$, not to the modulo-$4$ H13 phase. The amplitude scale is still the same up to an absolute constant, but the phase constant matters for spacing.

3. **Vaaler page references remain uncertain.** A3 suggests page/equation anchors, but the local PDF extraction indicates Theorem 18 is around printed page 210, not page 200. A1 should perform the final page-level source audit before any page number enters the proof draft.

4. **Case A source inconsistency needs recording.** The Li--Yang TeX source contains a line with $M<T^{-7/16}$ in the Case A definition and later uses $M<T^{7/16}$. This should be checked against the PDF or author source before using the condition in a formal subrange table.

5. **Toy computations are not endpoint evidence.** A3's small $X=16$ and $X=100$ examples are formula-debugging checks only. They should not be described as supporting an asymptotic claim.
