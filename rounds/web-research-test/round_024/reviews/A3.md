## Possible errors or hidden assumptions

* **A2's near-collision band size** -- the non-oscillatory band is $0<|N|\lesssim D^4/X$.  At endpoint $D\asymp X^{1/2}$ this is $\lesssim X$.  The heuristic then multiplies this interval length by the average unweighted count per $N$ to obtain $X^{9/4}$ unweighted terms.  The average count $X^{5/4}$ is derived from (total terms)/(max $N$) = $H_D^4 D^4 / (H_D D^3) = H_D^3 D$, which at endpoint gives $X^{3/4+1/2}=X^{5/4}$.  That arithmetic is correct under the uniform model.  **The hidden assumption is uniformity**; the map $(h_i,d_i)\mapsto N$ may cluster strongly, and the actual absolute weighted mass could be far from the model.
* **A2's denominator-paired parameterization**: writing $d_1=ag,\ d_3=bg$ leads to $c$ with $h_1-h_2=ac$ and $h_4-h_3=bc$.  The weighted sum involves denominators $|h_1|^{-1}|h_1-ac|^{-1}|h_3|^{-1}|h_3+bc|^{-1}$.  The analysis does not yet treat the regime where $ac\approx h_1$ or $bc\approx h_3$, which can produce amplified weight; these cases should be separated before a proof-grade bound is asserted.
* **M9-Pair-R24 real-weight assumption** -- the paired formulas require $w_D$ real.  If future numerical tests use complex weights or non-symmetric cutoffs, the formulas silently become wrong.  The proof draft explicitly notes that restriction, but it is an implementation pitfall.
