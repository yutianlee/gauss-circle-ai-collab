## Possible errors or hidden assumptions

1. **A3's endpoint extension for $\Phi$ appears wrong.** A3 states that $\Phi(0)=\Phi(1)=1$ by continuous extension. For
$$
\Phi(u)=\pi u(1-u)\cot(\pi u)+u,
$$
one has $\Phi(0^+)=1$, but
$$
\Phi(1^-)=0,
$$
because $\pi u(1-u)\cot(\pi u)\to -1$ and $u\to1$. This does not affect coefficients with $1\le h\le H$, since $h/(H+1)<1$, but it must be corrected in the implementation notes.

2. **A2's fourth-moment diagonal acceptability is conditional, not proved.** The statement that the diagonal is acceptable depends on an $L^2$ square-root-type control of $S_h$ or equivalent cancellation. It should not be promoted to a lemma until the complete diagonal/semi-diagonal taxonomy is written.

3. **A2's CRI statistic is not a bound.** It is useful only if it is paired with estimates for $|\Sigma_1|^2$ and $|\Sigma_3|^2$ and if the sign of $\Re(\Sigma_1\overline{\Sigma_3})$ is favorable. A small absolute value of $\mathcal S_{CRI}$ would not help; a negative real part would actively hurt.

4. **The $\mathcal M_2$ shifted-$F$ theorem-extension problem remains open.** The phase
$$
F_{\rho,D}(z)=\frac1{4z}+\frac{\rho D}{4X}
$$
has the same derivative nondegeneracy as $1/(4z)$, but theorem applicability is not automatic. A theorem must permit the parameter-dependent additive shift, the actual Vaaler weights, odd-frequency restrictions or equivalent shifted-frequency formulation, smooth $d$ weights, and endpoint height $H_D\asymp DX^{-1/4}$.

5. **Operator-norm blindness is bounded-scope.** Q1-Spectral proves that if $\chi_4$ enters only as a diagonal unitary conjugation $U^*KU$, unitarily invariant norms and pure cyclic traces cannot exploit it. It does not rule out direct signed bilinear forms, open-path moments, non-conjugacy traces, residue interference, or a new signed spacing theorem.

6. **Numerical tests must not be overinterpreted.** R5 tests verify constants and edge cases. Raw-versus-paired M9 tests verify algebra. Q1 tests verify norm invariance. BSOS/CRI/H13 tests can falsify specific finite-statistic routes. None of these numerical checks can prove the endpoint exponent by themselves.
