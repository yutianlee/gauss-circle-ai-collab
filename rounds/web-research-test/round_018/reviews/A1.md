## Claims that look correct

1. **The global status is correctly conservative.** Both agents correctly state that no new Gauss circle exponent is proved and that `M9`, not the Vaaler residual, remains the active analytic bottleneck.

2. **`Q1-Spectral-R18` is correct under its stated hypothesis.** If the character enters as diagonal conjugation on the odd-denominator Hilbert space, then unitarily invariant norm estimates cannot see it:

$$
\|U^*KU\|=\|K\|.
$$

This applies to operator norm, Frobenius norm, singular spectrum, absolute-value Schur-type bounds, and pure conjugacy-invariant cyclic traces.

3. **`H12-R18` is correct in the narrow cyclic-trace form.** For pure traces,

$$
\operatorname{Tr}((U^*KU)^m)
=
\operatorname{Tr}(K^m).
$$

This blocks only trace methods that are literally conjugacy-invariant. It does not block open-path moments, signed bilinear forms, cross-residue statistics, or methods that evaluate the vector-specific signed form before norm extraction.

4. **The Vaaler coefficient conjugacy is correct.** Since

$$
\alpha_{h,H}
=
-\frac{\Phi(|h|/(H+1))}{2\pi i h}
$$

with real $\Phi$, one has

$$
\alpha_{-h,H}=\overline{\alpha_{h,H}}.
$$

Any implementation using $\alpha_{-h,H}=-\overline{\alpha_{h,H}}$ is wrong.

5. **A3's `R5-Full` strategy is correct in principle.** The residual is controlled by the Fejer kernel bound

$$
K_H(t)\ll \min\left(H,\frac1{H\|t\|^2}\right),
$$

hence

$$
\frac1H K_H(t)\ll \min\left(1,\frac1{H^2\|t\|^2}\right).
$$

For $d\asymp D$, $H\asymp DX^{-1/4}$, and $\Delta=D/H\asymp X^{1/4}$, the first residual leg reduces to a product-counting estimate of the shape

$$
\sum_{n\asymp X}\tau(n)W_\Delta(X-n)
\ll_\epsilon X^{1/4+\epsilon}.
$$

The shifted legs reduce similarly after writing $\ell=4m-\rho$.

6. **The Li--Yang black-box import remains unavailable.** The arXiv abstract confirms the Bombieri--Iwaniec / first-spacing / Huxley-second-spacing framework, but the theorem still proves the Li--Yang exponent $\theta^*=0.314483\ldots$, not the endpoint $\theta=1/4$. The TeX source labels listed by A3 are useful, but endpoint applicability to `M9` remains a theorem-level gap.

## Possible errors or hidden assumptions

1. **A2 overstates uniform size of Vaaler coefficients.** The statement $|\alpha_{h,H}|\asymp 1/|h|$ is not uniformly true up to $h=H$. It is safe to use

$$
|\alpha_{h,H}|\ll \frac1{|h|},
$$

and

$$
\sum_{1\le |h|\le H}|\alpha_{h,H}|\ll\log H.
$$

But near the cutoff, $\Phi(h/(H+1))$ is small. This does not damage the Cauchy--Schwarz diagnostic, but the wording should be corrected.

2. **A2's weighted kernel must not be silently interchanged with an unweighted or $|\alpha|^2$ kernel.** A2 chooses weighted Cauchy--Schwarz, giving a kernel with $|\alpha_h|$. A3 later writes a Cauchy--Schwarz variant involving $|\alpha_h|^2$. Both can be legitimate if derived separately, but they produce different kernels and different BSOS normalizations. Round 19 must freeze exactly one normalization for each diagnostic test.

3. **A3 has an exact-resonance wording error in `R5`.** At an integer argument, $K_H(0)=H+1$, so

$$
\frac1H K_H(0)=1+\frac1H\le 2.
$$

A3's sentence saying $\frac1H K_H(t)\asymp H+1$ is wrong. The intended cap $W_\Delta(0)=1$ is correct up to an absolute constant, but the proof text must not contain the inflated resonance size.

4. **A3's small-scale R5 example is not a dyadic-block test as written.** For $X=16$, $D=4$, the dyadic interval $[D,2D)$ contains $d=4,5,6,7$, not only $d=4$. A toy single-resonance check is acceptable if labeled as such, but not as a representative dyadic-block computation.

5. **The H13 sign convention still needs correction.** With the convention

$$
\widehat f(\xi)=\int f(u)e(-\xi u)\,du,
$$

Poisson modulo $4$ gives a term like

$$
\frac{i}{2}\sum_k\chi_4(k)\int w_D(u)e\left(\frac{hX}{u}-\frac{k u}{4}\right)\,du.
$$

The stationary point occurs for $k=-m<0$, giving phase

$$
\frac{hX}{u}+\frac{m u}{4},
\qquad
u_0=2\sqrt{\frac{hX}{m}},
\qquad
\phi(u_0)=\sqrt{hXm}.
$$

If the phase is written as $hX/u-mu/4$, then the active stationary index has the opposite sign. A3's output uses both conventions loosely. This must be fixed before any post-H13 kernel is trusted.

6. **A2's "dual character erased" claim depends on the first post-H13 step.** It is correct only if the first spacing step genuinely produces a conjugated kernel $\widetilde U^*\widetilde K\widetilde U$. If a non-conjugacy statistic is constructed before norm extraction, `Q1` does not apply.
