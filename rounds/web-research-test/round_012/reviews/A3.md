## Possible errors or hidden assumptions

* **Vaaler coefficient sign convention.** A1 correctly notes that Montgomery--Vaughan approximates $1/2-\{x\}$, so the sign for $\psi(t)=\{t\}-1/2$ is negative. However, the exact constant in the Fejer majorant may change if a different normalization of $K_H$ is used (some sources define $K_H(t)=(H+1)^{-1}(\sin\,\pi(H+1)t/\sin\pi t)^2$ without the $2H+2$ denominator). The final residual bound $|R_H(t)|\le \frac1{2H+2}K_H(t)$ is plausible but must be source-matched; otherwise the factor $1/H$ in the R5 estimates might be off by a constant, which is harmless for exponents but should be exact for reproducibility.
* **Dyadic weight handling in R5.** A1 uses $|w_D(d)|\le1$ and groups by $n=md$. The grouping assumes that $m=m(d)$ is the nearest integer; for $X/d$ halfway between two integers, either choice works. However, when $X$ is not an integer, the distance $\|X/d\|$ equals $|X-md|/d$ only if $m$ is the nearest integer. This holds, but one must check that the classification "$m$ nearest to $X/d$" does not create double counting when both nearby integers are equally close; a small tie-breaking rule suffices.
* **Shifted second leg: positivity of $\ell$.** A1 writes $\ell=4m-\rho$ and claims $\ell\asymp X/d>0$. For $d$ near $X$ (possible only when $D\approx X$), $\ell$ could be $O(1)$; however, $D\le X^{1/2}$ by construction, so $X/d\ge X^{1/2}\gg 1$ for large $X$. The finite small-$X$ cases can be absorbed into the implicit constant.
* **A2's shifted-trace orthogonality.** The operator $T_q$ is not unitary, and the expression $\operatorname{Tr}(T_q(U^*KU)^p)$ is not Hermitian, so bounding it via standard spectral methods is delicate. The proposal does not yet constitute an estimate.
* **A2's C3-Affine connection to M9.** The parity collapse is proved for a stylized half-integer lattice model; its direct relevance to the actual dual variables arising from H13 or odd-lattice transforms is plausible but not yet formally linked. The claim that this "blocks coefficient-level parity gains for direct translation A-process models" is therefore a diagnostic, not a proven obstruction.
